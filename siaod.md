- Примечание. Для ввода бинарного дерева можно использовать функцию TreeInsert
```Tree* TreeInsert(Tree* t, int x) {
    if (t == nullptr) {
        t = new Tree;
        t->v = x;
        t->l = nullptr;
        t->r = nullptr;
        return t;
    }
    if (x < t->v)
        t->l = TreeInsert(t->l, x);
    else
        t->r = TreeInsert(t->r, x);
    return t;
}```
